# @full-stack-ds/figma-plugin

Figma desktop plugin scaffold for materializing Full Stack DS component contracts into Figma documentation pages, components, variants, and component properties.

The plugin consumes descriptors generated by `@full-stack-ds/codegen --target=figma` under `src/generated/components`. Those descriptors are transfer artifacts derived from `ComponentIR`; they are not source of truth.

## Current scope

- Reads the generated Figma component registry.
- Creates a documentation page and a components page.
- Scaffolds one documentation frame per contract descriptor.
- Creates placeholder component frames named from the contract and annotates them with plugin data.
- Leaves final Figma API materialization details in `TODO.md`.

## Build

```bash
pnpm --filter @full-stack-ds/figma-plugin run build
```

Load `manifest.json` from this package in Figma Desktop during development.
